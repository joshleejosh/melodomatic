PITCH CODE

Pitch values can be encoded with octave and accidental info to step outside their prescribed scale pitches.

A - or + before the pitch value raises or lowers the octave.

A - or + after the pitch value flats or sharps the note.

Don't leave any spaces between the modifiers and the value, or they'll be read as separate notes!

 
 
:voice PITCHY
 
.duration 1
 
.velocity 64
 
.pitch $loop
